# GilViewer #

GilViewer is a C++ library which aims at helping developpers to add 2D raster and vector viewer functionalities in their applications. It is currently based on [wxWidgets](http://www.wxwidgets.org/) for the GUI and on [Boost/GIL](http://opensource.adobe.com/wiki/display/gil/Generic+Image+Library) for image handling.

GilViewerApp, a sample application is also available in the repository.
