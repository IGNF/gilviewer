GilViewer

GilViewer is an open source 2D viewer (raster and vector) based on Boost
GIL and wxWidgets.

GilViewer is also based on TinyXML (http://www.grinninglizard.com/tinyxml/)
and Shapelib (http://shapelib.maptools.org/).
